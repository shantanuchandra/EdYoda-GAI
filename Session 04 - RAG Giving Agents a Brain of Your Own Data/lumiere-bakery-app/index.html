How this demo wires up to Session 4

Every concept from S4 is live in this app:

  • Build A — Feed the brain (CAG): small doc loaded whole into the prompt.
  • Build B — Make it cite: every answer ends with (Source: …).
  • Build C — Make it refuse: ask out-of-scope, watch it decline cleanly.
  • Build D — Tune the recall: swap CLEAN vs MESSY policy doc tiles, see the same query land vs miss.
  • Build E — Stress & ship: the bot is the shipped agent.
  • The Dial in motion: CAG / RAG toggle in admin.
  • S5 bridge — Agentic RAG: location tool + visible thinking shows the agent deciding what to retrieve.
